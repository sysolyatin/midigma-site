MIDI to grandMA2

Удобная работа с MIDI‑контроллерами в grandMA2 - это возможно!



Скачать для Windows
Простая настройка

Не нужно запоминать MIDI ноты и вводить их вручную, не нужно делать лишние настройки в пульте (только включить удалённое управление).

Независимость от шоу-файла в grandMA и самого пульта

Приложение можно использовать с любым пультом семейства grandMA2: от onPC до Full-Size.

Свободный MIDI вход

Наше приложение не использует MIDI вход в пульте GrandMA, значит его можете использовать вы, например, для приёма MIDI TimeCode.

Работа с любыми MIDI контроллерами

Если Windows может распознать ваш контроллер, значит он может работать с этим приложением!

Сохранение настроек

Все сделанные настройки и добавленные команды для grandMA можно сохранить и использовать в любом шоу-файле.

Поддержка всех MIDI команд

Приложение поддерживает не только приём MIDI нот, но и сигналы об изменении контролов (движения фейдеров) и энкодеров.

Одно приложение для решения задачи

Не нужно использовать цепочку из нескольких приложений для управления grandMA с помощью MIDI контроллера.

Полное разделение

grandMA не будет ничего знать о вашем MIDI контроллере, но будет с него управляться - это избавит ваше шоу от лишних ошибок.

Вопросы и ответы

С помощью MIDI контроллера можно изменять значение фейдера экзекьютора, нажмать кнопки экзекьютора, нажимать любую функциональную клавишу пульта (hardkey), изменять значене колес атрибутов с помощью физического энкодера, выбирать страницу для колес атрибутов (Dimmer, Position, Gobo и так далее), а так же в виде команды выполнять любое действие в grandMA2, если оно поддерживается комадной строкой grandMA2.
В первую очередь, на grandMA необходимо войти в сессию с адресом, который будет доступен на компьютере, где запущено наше приложение. Если вы используете onPC версию пульта - наше приложение можно устанавливать на один компьютер с приложением grandMA.
В глобальных настройках grandMA необходимо активировать Remotes управление. Без этого наше приложение не сможет связаться с пультом.
Для этого нажмите клавишу Setup и выберите пункт Global Settings

В открывшемся окне необходимо активировать Remotes

После этого необходимо открыть приложение MIDI to grandMA и можно подключаться.

В разделе Connect to grandMA 2 необходимо заполнить поля и нажать кнопку подключения, либо переподключения:
IP address - сетевой адрес пульта. Если приложение запущено на компьютере, где запущено приложение - то адрес - 127.0.0.1
Login - пользователь на пульте grandMA
Password - пароль от пользователя пульта grandMA

Рекомендуем создать отдельного пользователя в grandMA2 для подключения к приложению.

Системные требования: Операционная система Windows 7 или Windows 10, 50 МБ свободного места на диске, .NET Framework 4.7, наличие активного сетевого линка (для поднятия сессии в grandMA 2, можно даже просто 127.0.0.1).
